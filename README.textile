h1. node_mDNS

h2. Synopsis

node_mDNS is a "node.js":/ry/node add-on. It adds support for mDNS service discovery, also known as zeroconf or bonjour to "node.js":/ry/node.

h2. Build

Currently only tested on Mac OS X. Other platforms will require some tweaking of wscript.

bc. node-waf configure build
sudo node-waf install


h2. Examples

p. Announce a tcp server on port 4321 with service type 'bacon'

bc.. var service = mDNS.createService('bacon', 4321, mDNS.TCP);

service.announce();
setTimeout(function() { service.shutdown(); }, 10000);
